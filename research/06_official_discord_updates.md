## Source discipline

Discord is useful for fast updates, but authority matters:

- **Treat as official:** signed Host Scientific Committee / ISC / International Jury notices, posts from clearly identified organizers, the IOAI website, and the `IOAI-official` GitHub organization.
- **Treat as provisional:** contestant answers, code attachments, claimed scores, interpretations of “planning/search,” model-runtime claims, and forwarded messages whose original author/role is unclear.
- **Never let chat override the task statement or current rules.** When they conflict, ask through the official clarification/appeal route.
